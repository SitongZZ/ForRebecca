# ForRebecca

## About

ForRebecca is Freddie and Rebecca’s execution workspace for building practical AI products that create real-world value in work and life.

**Mission:** Build and ship AI tools and content systems that can generate sustainable side income and compound into long-term net-wealth growth.

### Current Focus
- AI micro-tools for professionals (starting from audit/productivity pain points)
- Content engine for audience growth and lead generation
- Automation workflows to improve execution speed and consistency

### 2026 Target
- Build toward **£10,000/year** side income from AI projects.

## Working Principles
- Ship fast, validate with users, iterate weekly
- Prioritize practical value over overengineering
- Track decisions and memory in workspace files for continuity

## Core Workspace Files
- `MEMORY.md` → long-term strategic memory
- `memory/YYYY-MM-DD.md` → daily logs
- `USER.md` → user profile, preferences, goals
- `IDENTITY.md` → Rebecca identity and operating vibe
